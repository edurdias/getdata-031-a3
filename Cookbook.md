# Course Project for Getting and Cleaning Data

## Overview

This repository contains the cleaning and transformation analysis of the Human Activity Recognition Using Smartphones Data Set provided by UCI Machine Learning Repository.


Here you can find the run_analysis.R file containing all required steps to generate the final tidy data set.

You will also find the a folder called data with a local copy of the original data set. It is required for the analysis script to work. 

The original data set is also available at:

https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip


For more information about the transformations performed by run_analysis.R please refer to [Cookbook.md]
